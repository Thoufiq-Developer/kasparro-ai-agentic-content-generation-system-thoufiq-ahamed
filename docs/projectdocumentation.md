
# Multi-Agent Content Generation System

## Problem Statement
Design a modular agentic system that converts structured product data into machine-readable content pages.

## Solution Overview
The system uses multiple agents with single responsibilities to parse data, apply logic, and assemble JSON outputs.

## System Design
Pipeline:
Data Parser → Logic → Template → Page Assembler

## Conclusion
This project demonstrates production-style agent orchestration and modular AI system design.
