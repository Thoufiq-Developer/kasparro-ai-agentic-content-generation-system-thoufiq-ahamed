This project implements a modular, multi-agent automation system that transforms structured product data into clean, machine-readable JSON content through a well-defined orchestration pipeline. Each agent has a single responsibility—parsing input data, applying reusable content logic, and assembling final outputs—ensuring clarity, modularity, and extensibility. The system is intentionally backend-focused, with no frontend or prompt-only generation, and operates strictly on the provided data without external assumptions. Designed to reflect real production AI workflows, it demonstrates agent orchestration, reusable logic blocks, and template-driven generation suitable for scalable content and automation platforms.
